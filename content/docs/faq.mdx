# 💬 자주 묻는 질문

## 🙋 로그인은 꼭 해야 하나요?

네, 현재 버전은 Google 로그인으로 이용해요. 홈 화면에서 `무료로 시작하기`를 누르면 로그인 페이지가 열리고, `Google로 계속하기` 버튼으로 들어올 수 있어요.

![Google 로그인 안내](/images/example.png)

## 🗄️ 데이터는 어디에 저장되나요?

모든 프로젝트와 다이어그램은 서버에 자동 저장돼요. 편집기 상단의 `저장 중...` 배지가 `저장 완료`로 바뀌면 서버에 반영이 끝났다는 뜻이에요. 같은 계정으로 로그인하면 다른 기기에서도 바로 이어서 작업할 수 있어요.

## 🤔 AI가 제안한 다이어그램을 일부만 쓰고 싶어요

AI 응답은 JSON 스키마 전체를 보내기 때문에 자동 반영 시 다이어그램이 통째로 교체돼요. 일부만 적용하려면 응답을 참고해 필요한 테이블과 컬럼만 수동으로 추가하거나, JSON을 편집해 다시 `스키마 가져오기`로 넣어주세요.

## 🔗 엔티티나 컬럼을 삭제하면 관계도 사라지나요?

네, 엔티티를 삭제하면 연결된 관계도 함께 지워져요. 컬럼을 삭제할 때는 기본키나 외래키인지 여부를 확인하는 경고 문구가 먼저 나타나요. 실수로 지우지 않도록 항상 확인 창을 보여주니 안심하세요.

## 🤝 팀원과 어떻게 공유하나요?

상단 툴바의 `스키마 내보내기`에서 MySQL, PostgreSQL, Prisma, JPA 중 하나를 골라 복사하거나 다운로드하세요. 상대 팀에서는 동일한 형식을 `스키마 가져오기`로 불러오면 같은 구조를 확인할 수 있어요.

![스키마 내보내기 버튼 위치](/images/example.png)

## 🖼️ PNG 내보내기는 언제 되나요?

버튼은 이미 자리 잡고 있지만 아직 구현 중이에요. 업데이트가 배포되면 문서와 릴리즈 노트에 함께 안내할 예정이에요. 그전까지는 브라우저 캡처 기능을 활용해주세요.
