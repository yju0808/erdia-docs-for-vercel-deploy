# 👋 Erdia에 오신 걸 환영해요

Erdia는 **AI 채팅과 인터랙티브 캔버스**로 ERD를 빠르게 설계하고, MySQL, PostgreSQL, Prisma, JPA 코드로 바로 변환할 수 있는 툴이에요.

아이디어를 자연어로 말하면 즉시 다이어그램이 만들어지고, 드래그 앤 드롭으로 세밀하게 다듬은 뒤, 팀이 사용하는 형식으로 공유할 수 있습니다.

![랜딩 화면에서 살펴보기](/images/example.png)

---

## 🚀 5분 안에 시작하기

1. 홈 화면에서 **무료로 시작하기**를 누르고 Google 계정으로 로그인하세요. 인증이 끝나면 자동으로 대시보드로 이동해요.
2. `새 프로젝트` 버튼을 누르고 이름을 입력하세요. 이름은 중복 검사를 거쳐서 실수로 덮어쓸 걱정이 없어요.
3. 프로젝트를 만들면 즉시 편집기로 이동해요. 모든 변경사항은 자동으로 저장되니 마음 편히 작업할 수 있습니다.

<Callout type="warning">
로그인 화면의 **게스트 모드로 둘러보기** 버튼은 아직 준비 중이에요. 실사용은 Google 로그인으로 진행해 주세요.
</Callout>

## 🎨 ERD 설계하기

![ERD 편집 화면](/images/example.png)

### 2. 테이블 직접 그리기

- `엔티티 추가` 버튼을 누르거나 인터랙티브 캔버스에서 우클릭하세요
- 컬럼 추가, 타입 변경, 관계 연결 모두 가능해요

**컬럼 편집**
- 연필 아이콘으로 타입, 기본키, 외래키, NULL 여부를 설정하세요
- 삭제할 때는 영향 범위를 알려주는 경고가 먼저 나타나요

**관계 연결**
- 컬럼 옆 작은 원을 드래그해서 다른 테이블로 연결하세요
- 1:1, 1:N 같은 관계 타입을 선택하면 자동으로 선이 그려져요

**자동 정렬**
- 버튼 한 번으로 겹치는 테이블을 깔끔하게 정리할 수 있어요

<Callout type="success">
편집할 때마다 약 1초 뒤 서버에 자동 저장돼요. 다른 기기에서 같은 계정으로 로그인해도 가장 최근 상태를 바로 이어서 볼 수 있습니다.
</Callout>

## 🤖 AI로 빠르게 설계하기

하단 채팅창에 "온라인 쇼핑몰 ERD 만들어줘" 같은 자연어로 요청하면, AI가 전체 구조를 한 번에 만들어줘요.

![AI 채팅 응답 예시](/images/example.png)

**사용 팁**
- 하루 50회까지 무료로 사용할 수 있어요
- 기존 ERD를 수정하고 싶다면 "User 테이블에 email 컬럼 추가해줘" 처럼 구체적으로 요청하세요
- 일부만 적용하고 싶다면 응답을 참고해서 직접 편집하거나, 스키마 가져오기 기능을 사용하세요

<Callout type="warning">
AI가 새 구조를 반영하면 이전 다이어그램은 덮어써요. 중요한 버전은 먼저 내보내기 탭에서 SQL이나 Prisma 파일로 백업해 두세요.
</Callout>

## 📤 팀에 공유하기

완성된 ERD를 MySQL, PostgreSQL, Prisma, JPA 코드로 바로 변환할 수 있어요.

![스키마 내보내기](/images/example.png)

**내보내기 옵션**
- 외래키, 인덱스, 타임스탬프 포함 여부를 선택할 수 있어요
- 코드를 복사하거나 파일로 다운로드 받으세요
- 팀 규칙에 맞춰 필요한 옵션만 켜서 사용하면 돼요

<Callout type="info">
PNG 내보내기 버튼은 UI에 먼저 노출돼 있지만 아직 구현 전이에요. 이미지가 필요하면 브라우저 캡처 기능을 사용해주세요.
</Callout>
