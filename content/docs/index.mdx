# Erdia에 오신 것을 환영합니다

**Erdia**는 AI 기반의 차세대 ERD(Entity-Relationship Diagram) 디자인 플랫폼입니다. 복잡한 데이터베이스 설계를 자연어 대화만으로 시작할 수 있으며, 인터랙티브한 캔버스에서 실시간으로 수정하고, 원하는 데이터베이스 엔진에 맞는 SQL을 즉시 내보낼 수 있습니다.

## 🎓 누가 사용하나요?

- **개발자**: 프로젝트 초기 DB 설계를 빠르게 완료하고 싶은 백엔드 개발자
- **데이터 엔지니어**: 복잡한 데이터 모델을 시각화하고 관리해야 하는 전문가
- **학생**: 데이터베이스 과목을 수강하며 ERD 실습이 필요한 학생
- **기획자**: 개발자와 협업하기 위해 데이터 구조를 이해하고 싶은 기획자
