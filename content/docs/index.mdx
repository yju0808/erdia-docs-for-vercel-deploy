# 👋 Erdia에 오신 것을 환영합니다

Erdia는 대화형 AI 어시스턴트와 비주얼 캔버스를 결합해 데이터베이스 구조를 빠르게 설계할 수 있도록 도와주는 ERD(Entity-Relationship Diagram) 빌더입니다. 자연어 프롬프트와 드래그 앤 드롭을 자유롭게 오가며, 프로젝트별 다이어그램을 브라우저에 안전하게 저장하고 필요할 때마다 불러올 수 있습니다.

![Erdia 편집기 미리보기](/images/example.png)

## 🚀 지금 바로 시작하세요

1. **홈 화면에서 `시작하기` 버튼을 눌러 대시보드로 이동**하면 프로젝트 목록이 열립니다.
2. **`새 프로젝트`를 생성**하고 이름과 설명을 입력하면 빈 다이어그램이 준비됩니다.
3. **캔버스 툴바와 AI 어시스턴트를 활용**해 엔티티, 속성, 관계를 완성한 뒤 필요하면 스키마를 내보내세요.

<Callout type="info">
브라우저의 로컬 스토리지(localStorage)에 프로젝트와 다이어그램이 저장되므로, 같은 브라우저에서 언제든지 이어서 작업할 수 있습니다. 다른 기기에서 동일한 다이어그램을 열려면 JSON 또는 SQL로 내보낸 뒤 가져오기를 사용하세요.
</Callout>
