# 👋 Erdia에 오신 걸 환영해요

Erdia는 **테이블**(데이터가 담기는 표)과 **컬럼**(테이블 안의 칸)을 생각나는 대로 적어도, 자연스럽게 구조를 정리해 주는 도우미예요. AI에게 설명해도 되고 직접 끌어다 놓아도 돼서, 머릿속 그림을 바로 ERD(Entity-Relationship Diagram)로 옮길 수 있죠.

![랜딩 화면에서 살펴보기](/images/example.png)

## 🚀 5분 만에 한 바퀴 둘러보기

1. **홈에서 `무료로 시작하기` 누르기** – 로그인 화면으로 이동하고, Google 계정으로 들어오면 돼요.

   ![Google 로그인 버튼 위치](/images/example.png)

2. **대시보드에서 프로젝트 고르기** – 목록은 최근 수정 순으로 정렬돼요. 검색창에 키워드를 적으면 바로 필터링돼요.

   ![대시보드 검색과 카드 예시](/images/example.png)

3. **`새 프로젝트`로 빈 다이어그램 만들기** – 이름과 설명만 정하면 깨끗한 캔버스가 열려요.

   ![새 프로젝트 창 예시](/images/example.png)

4. **캔버스에서 테이블 배치하기** – 엔티티 카드 하나가 테이블을 의미해요. 더블클릭으로 이름을 바꾸고, 컬럼을 추가하면서 관계(relationship)를 연결해 보세요.

   ![캔버스에서 테이블 편집하기](/images/example.png)

5. **AI에게 손봐 달라고 부탁하기** – 하단 채팅 입력창에 "주문 테이블에 상태 컬럼 추가해 줘"처럼 말하면, 설명과 함께 JSON 스키마를 보내줘요.

   ![AI 채팅 응답 예시](/images/example.png)

6. **스키마 내보내기로 마무리하기** – MySQL, PostgreSQL, Prisma, JPA 중 원하는 형식으로 변환해 다운로드할 수 있어요.

   ![스키마 내보내기 탭](/images/example.png)

<Callout type="info">
Erdia는 프로젝트를 자동으로 클라우드에 저장해요. 상단 툴바 오른쪽에서 `저장 중...` → `저장 완료` 표시가 바뀌는 걸 확인해보세요.
</Callout>

## 🎯 Erdia가 주는 핵심 가치

- **실제 업무 언어로 설계하기** – 복잡한 용어 대신 자연어와 드래그 앤 드롭만으로 구조를 완성할 수 있어요.
- **관계가 눈에 보이는 편집기** – 엔티티 간 선이 어떻게 연결됐는지, 어느 컬럼이 외래키(다른 테이블과 연결하는 키)인지 바로 확인할 수 있어요.
- **팀과 이어지는 내보내기** – SQL DDL, Prisma 스키마, JPA 엔티티 클래스로 뽑아 개발 파이프라인에 연결하기 좋아요.
- **안심할 수 있는 자동 저장** – 편집할 때마다 서버에 저장돼서, 다른 기기에서도 같은 계정으로 로그인하면 바로 이어서 작업할 수 있어요.
