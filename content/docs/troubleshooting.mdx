# 🛠 문제 해결 가이드

다음은 사용자가 자주 겪는 상황과 해결 방법입니다. 문제가 반복되면 제일 아래의 **추가 조사 메모**를 참고해 지원팀에 전달해주세요.

## ❌ AI 응답이 오지 않을 때

- 증상: 채팅 입력 후 `Failed to fetch response from server.` 메시지가 나타납니다.
- 조치: OpenAI API 키가 서버 환경 변수 `OPENAI_API_KEY`에 설정되어 있는지 확인하세요. 네트워크 제한이나 일시적인 API 장애일 수도 있으니 잠시 후 다시 시도합니다.

## 🕳️ 빈 대시보드가 보일 때

- 증상: 기존 프로젝트가 사라졌거나 "프로젝트가 없습니다" 메시지만 표시됩니다.
- 조치: 브라우저의 로컬 스토리지가 초기화되었을 가능성이 큽니다. 시크릿 모드 또는 다른 브라우저에서는 저장 데이터가 공유되지 않으므로, 정기적으로 SQL/JSON 백업을 수행하세요.

## 🔁 자동 저장 상태가 `저장 중...`에서 멈출 때

- 증상: 편집기 상단 저장 표시가 `저장 중...`에서 사라지지 않습니다.
- 조치: 자동 저장은 변경 후 최대 1초까지 지연될 수 있습니다. 3초 이상 지속되면 페이지를 새로고침하고, 변경 사항이 반영되었는지 확인하세요.

## 🔐 프로젝트 이름이 저장되지 않을 때

- 증상: 새 프로젝트 생성 또는 수정 시 `중복된 이름` 경고 토스트가 발생합니다.
- 조치: 같은 이름의 프로젝트가 이미 존재합니다. 기존 프로젝트를 삭제하거나 이름을 구분할 수 있도록 수정하세요.

## 🔗 관계 연결이 실패할 때

- 증상: 속성을 드래그해도 관계가 생성되지 않고 연결 가이드가 계속 남아 있습니다.
- 조치: ESC 키를 눌러 연결 모드를 종료한 뒤 다시 시도하세요. 속성 선택 다이얼로그가 열린 상태에서는 다른 작업이 제한됩니다.

## 💬 채팅 사이드바가 닫히지 않을 때

- 증상: 모바일에서 채팅 드로어가 열린 상태로 고정되어 있습니다.
- 조치: 우측 상단 X 버튼을 누르거나 하단 입력창 외부를 탭해 닫을 수 있습니다. 화면 회전을 시도하면 레이아웃이 초기화됩니다.

## 📝 추가 조사 메모

- PNG 내보내기 버튼은 현재 구현되지 않았습니다. 향후 업데이트에서 기능을 제공할 예정입니다.
- 다국어(한국어 외) UI 지원 요청이 증가하고 있습니다. 향후 다국어 라벨링을 고려해 주세요.
- 다이어그램 JSON 내보내기/가져오기 전용 버튼을 제공하면 버전 관리를 쉽게 할 수 있다는 피드백이 있습니다.
