# 🛠 문제 해결 가이드

궁금한 점이 생기면 아래에서 비슷한 사례를 찾아보세요. 화면에 보이는 버튼이나 문구를 함께 적어 두었으니 따라 하기 쉬울 거예요.

## ❌ AI 응답이 오지 않을 때

![AI 오류 메시지 예시](/images/example.png)

- 채팅 입력 후 `Failed to fetch response from server.`라는 문구가 보이면, 잠시 후 다시 시도하거나 인터넷 연결을 확인하세요.
- 우측 하단 말풍선 버튼을 눌러 사이드바를 닫았다가 다시 열고, 새 메시지를 보내면 재시도할 수 있어요.

> 🔧 **개발자용**: 서버 환경 변수 `OPENAI_API_KEY`가 비어 있거나 외부 네트워크가 차단되면 위 오류가 발생해요. 로그에서 500 응답을 확인하고 키를 설정한 뒤 서버를 재시작하세요.

## 🕳️ 대시보드가 비어 있을 때

![비어 있는 대시보드 안내 카드](/images/example.png)

- 검색창에 입력한 키워드가 남아 있는지 확인하고, 텍스트를 모두 지워보세요.
- 페이지 상단의 `새 프로젝트` 버튼이 보이면, 새로고침하거나 잠시 후 다시 접속해 프로젝트 목록을 불러오세요.
- 여전히 카드가 없다면 로그아웃 후 다시 로그인해 세션을 갱신해보세요.

> 🔧 **개발자용**: `/api/projects` 요청이 401이라면 토큰이 만료된 상태예요. 백엔드 로그에서 인증 미들웨어 에러를 확인하고 토큰 갱신 엔드포인트가 정상 동작하는지 점검하세요.

## 🔁 자동 저장이 `저장 중...`에서 멈출 때

![저장 상태 배지 위치](/images/example.png)

- 저장 배지는 최대 1~2초 뒤 `저장 완료`로 바뀌어요. 잠시 기다려보세요.
- 5초 이상 그대로라면 인터넷 연결을 확인하고, `대시보드` 버튼으로 나갔다가 다시 들어와 최신 상태가 반영됐는지 확인하세요.
- 계속 반복되면 로그아웃 후 다시 로그인해 토큰을 갱신하세요.

> 🔧 **개발자용**: `/api/projects/{id}/diagram` PATCH 요청이 실패하면 저장 배지가 유지돼요. 서버 로그에서 500 응답을 확인하거나, 브라우저 개발자 도구 네트워크 탭을 점검하세요.

## 🔐 프로젝트 이름이 저장되지 않을 때

![프로젝트 이름 입력 창](/images/example.png)

- 새 프로젝트 창 또는 수정 창에서 이름을 비워두면 "이름을 입력하세요" 토스트가 보여요.
- 이미 존재하는 이름이라면 "중복된 이름" 토스트가 나타나니, 이름 뒤에 날짜나 팀명을 덧붙여 구분해 주세요.

## 🔗 관계 연결이 되지 않을 때

![관계 연결 가이드 배지](/images/example.png)

- 컬럼에서 선을 드래그한 뒤, 타깃 엔티티 위에서 놓였는지 확인하세요. 정확히 카드 위에서 놓아야 연결돼요.
- 연결 중이라면 상단 배지가 떠 있어요. ESC 키를 눌러 연결 모드를 취소한 뒤 다시 시도해 보세요.
- 속성 선택 창이 열린 상태에서는 다른 작업이 잠시 제한되니, 창을 닫은 뒤 다시 시도하면 돼요.

## 💬 채팅 사이드바가 닫히지 않을 때

![채팅 사이드바 닫기 버튼](/images/example.png)

- 사이드바 오른쪽 위 `X` 버튼을 누르거나, 하단 입력창 바깥을 클릭하면 닫혀요.
- 모바일에서는 화면 오른쪽을 살짝 스와이프하거나, 하단 플로팅 말풍선 버튼을 눌러 드로어를 닫을 수 있어요.

## 📝 추가로 기억해두면 좋은 것들

- `PNG 내보내기` 버튼은 아직 준비 중이에요. 이미지가 필요하면 브라우저 캡처 기능을 사용해주세요.
- 다국어 UI는 순차적으로 추가할 예정이에요. 필요한 언어가 있다면 고객 지원으로 알려주세요.
- JSON 내보내기/가져오기 전용 버튼을 준비 중이에요. 스키마 백업이 잦다면 [support@erdia.app](mailto:support@erdia.app)으로 의견을 보내주세요.
