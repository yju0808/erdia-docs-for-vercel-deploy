# 🛠 문제 해결 가이드

궁금한 점이 생기면 아래에서 비슷한 사례를 찾아보세요. 화면에 보이는 버튼이나 문구를 함께 적어 두었으니 따라 하기 쉬울 거예요.

## ❌ AI 응답이 오지 않을 때

![AI 오류 메시지 예시](/images/example.png)

**증상:**
- 채팅 입력 후 `Failed to fetch response from server.`라는 문구가 표시됨
- 전송 버튼이 흐려진 채로 응답이 오지 않음

**해결 방법:**

1. **잠시 후 재시도**
   - 네트워크 연결을 확인하고 1-2분 뒤 다시 시도해보세요

2. **사이드바 재시작**
   - 우측 하단 말풍선 버튼을 눌러 사이드바를 닫았다가 다시 열어보세요
   - 새 메시지를 보내면 재시도할 수 있어요

3. **한도 확인**
   - Free 플랜은 하루 50회로 제한돼요
   - 한도를 초과했다면 다음 날까지 기다려야 해요

<Callout type="info">
팀 또는 관리자에게 OpenAI 연결 상태를 확인해 달라고 요청해 보세요. 인증 키가 만료됐거나 외부 네트워크가 차단된 경우 같은 오류가 발생할 수 있어요.
</Callout>

## 🕳️ 대시보드가 비어 있을 때

![비어 있는 대시보드 안내 카드](/images/example.png)

**증상:**
- 로그인 후 대시보드에 프로젝트 카드가 하나도 없음
- "프로젝트가 없습니다" 메시지만 보임

**해결 방법:**

1. **검색 필터 확인**
   - 검색창에 입력한 키워드가 남아 있는지 확인하세요
   - 텍스트를 모두 지워서 전체 목록을 표시하세요

2. **페이지 새로고침**
   - F5를 누르거나 브라우저 새로고침 버튼을 클릭하세요
   - 잠시 후 다시 접속해 프로젝트 목록을 불러오세요

3. **로그아웃 후 재로그인**
   - 여전히 카드가 없다면 로그아웃 후 다시 로그인해 보세요
   - 인증 정보가 새로 고쳐지면서 정상적으로 로드될 수 있어요

<Callout type="warn">
오랫동안 로그인한 상태였다면 인증 정보가 만료됐을 수 있어요. 문제가 계속되면 관리자에게 문의하세요.
</Callout>

## 🔁 자동 저장이 `저장 중...`에서 멈출 때

![저장 상태 배지 위치](/images/example.png)

**증상:**
- 편집기 상단 툴바의 저장 배지가 `저장 중...`에서 `저장 완료`로 바뀌지 않음
- 5초 이상 저장 중 상태가 계속됨

**해결 방법:**

1. **잠시 대기**
   - 일반적으로 1~2초 안에 `저장 완료`로 바뀌어요
   - 네트워크가 느린 경우 최대 5초까지 걸릴 수 있어요

2. **저장 상태 확인**
   - `대시보드` 버튼으로 나갔다가 다시 프로젝트를 열어보세요
   - 카드의 최근 수정 시간이 업데이트됐는지 확인하세요

3. **로그아웃 후 재로그인**
   - 계속 반복되면 로그아웃 후 다시 로그인해보세요
   - 편집 내용을 미리 `스키마 내보내기`로 백업해 두는 것을 추천해요

<Callout type="warn">
브라우저가 일시적으로 오프라인 상태였을 수 있어요. 새로고침 후 다시 편집해보고, 문제가 반복되면 관리자에게 서버 로그를 확인해 달라고 요청하세요.
</Callout>

## 🔐 프로젝트 이름이 저장되지 않을 때

![프로젝트 이름 입력 창](/images/example.png)

**증상:**
- 프로젝트 생성/수정 시 "이름을 입력하세요" 토스트가 표시됨
- "중복된 이름" 토스트가 나타남

**해결 방법:**

| 오류 메시지 | 원인 | 해결책 |
|----------|------|--------|
| "이름을 입력하세요" | 이름 필드가 비어 있음 | 프로젝트 이름을 입력하세요 (필수) |
| "중복된 이름" | 동일한 이름의 프로젝트가 이미 존재 | 이름 뒤에 날짜나 팀명을 추가하세요 |

**예시:**
- ❌ `Shopping Mall` → ✅ `Shopping Mall 2024`
- ❌ `User DB` → ✅ `User DB - Team A`

## 🔗 관계 연결이 되지 않을 때

![관계 연결 가이드 배지](/images/example.png)

**증상:**
- 컬럼 옆 작은 원을 드래그했지만 관계가 생성되지 않음
- 연결 가이드 배지만 보이고 선택 창이 열리지 않음

**해결 방법:**

1. **정확한 위치에 놓기**
   - 컬럼에서 선을 드래그한 뒤, 타깃 엔티티 카드 위에서 놓으세요
   - 카드 바깥에서 놓으면 연결이 취소돼요

2. **연결 모드 초기화**
   - 상단에 파란색/초록색 가이드 배지가 보이면 연결 중이에요
   - ESC 키를 눌러 연결 모드를 취소한 뒤 다시 시도하세요

3. **다른 창 닫기**
   - 속성 선택 창이나 다른 모달이 열려 있으면 작업이 제한돼요
   - 모든 창을 닫은 뒤 다시 드래그해보세요

<Callout type="info">
연결 시 출발점(파란색)과 도착점(초록색)이 명확히 구분돼요. 가이드 메시지를 읽으면 어느 방향으로 드래그해야 하는지 알 수 있어요.
</Callout>

## 💬 채팅 사이드바가 닫히지 않을 때

![채팅 사이드바 닫기 버튼](/images/example.png)

**증상:**
- 사이드바가 화면에 계속 떠 있음
- X 버튼을 눌러도 사라지지 않음

**해결 방법:**

**데스크톱:**
- 사이드바 오른쪽 위 `X` 버튼 클릭
- 또는 하단 우측 말풍선 아이콘을 다시 클릭

**모바일:**
- 화면 오른쪽을 왼쪽으로 스와이프
- 또는 하단 플로팅 말풍선 버튼을 눌러 드로어를 닫으세요
