# 주요 기능

Erdia는 데이터베이스 설계를 쉽고 빠르게 만드는 강력한 기능들을 제공합니다.

## 🎯 핵심 기능

### 1️⃣ AI 채팅 어시스턴트

**자연어로 ERD를 생성합니다**

AI 어시스턴트와 대화하듯이 데이터베이스 구조를 설명하면, 자동으로 ERD가 생성됩니다.

```
💬 사용자: "쇼핑몰 ERD 만들어줘. User, Product, Order 테이블이 필요해"

🤖 AI 어시스턴트:
쇼핑몰 ERD를 생성했습니다!

- User 테이블: 고객 정보 저장
- Product 테이블: 상품 정보 저장  
- Order 테이블: 주문 정보 저장
- 관계: User가 여러 Order를 가질 수 있고, Order는 여러 Product를 포함할 수 있습니다

[ERD가 캔버스에 자동 생성됨]
```

**AI가 자동으로 수행하는 작업:**
- ✅ 테이블 이름, 컬럼, 데이터 타입 추출
- ✅ 엔티티 간 관계(1:1, 1:N, N:M) 파악
- ✅ Primary Key, Foreign Key 자동 설정
- ✅ 한국어 설명을 이해하고 JSON 스키마로 변환

<Callout type="info">
**무료 플랜**: 하루 50회까지 AI 채팅을 사용할 수 있습니다.
</Callout>

### 2️⃣ 인터랙티브 캔버스 편집

**AI가 생성한 ERD를 실시간으로 수정합니다**

직관적인 인터페이스로 ERD를 자유롭게 편집할 수 있습니다.

**지원하는 편집 기능:**
- 🔹 엔티티 추가/삭제
- 🔹 속성(컬럼) 추가/수정/삭제
- 🔹 관계 타입 변경 (1:1, 1:N, N:M)
- 🔹 Primary Key, Foreign Key 설정
- 🔹 Nullable, Unique 제약조건 설정
- 🔹 노드 위치 자유롭게 이동
- 🔹 자동 레이아웃 정렬

<Callout type="success">
**실시간 미리보기**: 모든 변경사항이 즉시 캔버스에 반영됩니다
</Callout>

### 3️⃣ 다양한 DB 엔진 지원

**완성된 ERD를 SQL로 변환합니다**

지원 데이터베이스:
- ✅ **MySQL** (InnoDB 엔진, 인덱스 최적화)
- ✅ **PostgreSQL** (JSONB, Array, Custom Types)
- 🔜 **MariaDB** (준비 중)
- 🔜 **SQLite** (준비 중)

**생성되는 SQL 파일에 포함되는 내용:**
- `CREATE TABLE` 문
- `PRIMARY KEY`, `FOREIGN KEY` 제약조건
- 인덱스 생성
- Timestamp 필드 (`created_at`, `updated_at`)
- DB별 최적화된 데이터 타입

## 🛠 부가 기능

### 실시간 자동 저장

모든 변경사항이 자동으로 클라우드에 저장됩니다.

- ✅ 1초마다 자동 저장
- ✅ 브라우저를 닫아도 작업 내용 보존
- ✅ 저장 상태 실시간 확인

<Callout type="warning">
인터넷 연결이 필요합니다. 오프라인에서는 저장되지 않습니다.
</Callout>

### 다크 모드 지원

밝은 화면과 어두운 화면을 자유롭게 전환할 수 있습니다.

- 🌙 장시간 작업 시 눈의 피로 감소
- 🎨 개인 취향에 맞는 테마 선택

### 다이어그램 내보내기

다양한 형식으로 ERD를 내보낼 수 있습니다.

| 형식 | 용도 | 설명 |
|------|------|------|
| **PNG 이미지** | 문서화 | 다이어그램을 이미지로 저장 |
| **JSON 파일** | 백업/재사용 | ERD 데이터를 백업하거나 다른 프로젝트에서 재사용 |
| **SQL 파일** | DB 배포 | MySQL, PostgreSQL DDL 스크립트 다운로드 |

### 다이어그램 가져오기

백업한 ERD를 불러와서 작업을 재개할 수 있습니다.

- 📥 JSON 파일 업로드
- 🔄 기존 ERD 상태 복원
- 🤖 AI 채팅으로 추가 수정

### 키보드 단축키

빠른 작업을 위한 단축키를 제공합니다.

| 단축키 | 기능 |
|--------|------|
| `Delete` | 선택한 엔티티 또는 관계 삭제 |
| `Esc` | 다이얼로그 닫기 |
| `드래그 & 드롭` | 엔티티 이동 및 관계 연결 |
