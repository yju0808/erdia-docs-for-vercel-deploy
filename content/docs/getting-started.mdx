# 🚀 Erdia 빠른 시작 가이드

처음 오셨나요? 아래 순서대로 따라 하면 **5분 안에 첫 ERD를 완성**할 수 있어요. 설명은 실제 화면 동선과 동일하게 구성했어요.

## 1. 🔐 Google 계정으로 로그인하기

홈 화면에서 `무료로 시작하기`를 누르면 로그인 페이지가 열려요. 상단에 브랜드 로고와 안내 문구가, 중앙에는 큼직한 `Google로 계속하기` 버튼이 있어요. 버튼을 누르면 구글 인증 창이 뜨고, 완료되면 자동으로 Erdia로 돌아와요.

![Google 로그인 화면](/images/example.png)

<Callout type="warning">
`게스트 모드로 둘러보기` 버튼은 아직 준비 중이라 눌러도 이동하지 않아요. 정식으로 체험하려면 Google 로그인을 이용해주세요.
</Callout>

로그인 이후에는 "세션을 준비하고 있어요"라는 짧은 안내가 나타나고, 토큰 갱신이 끝나면 바로 대시보드로 이동해요.

## 2. 🗂 대시보드 훑어보기

대시보드는 프로젝트를 한눈에 볼 수 있는 홈이에요. 상단에는 로고와 랜딩으로 돌아가는 버튼, 우측에는 로그아웃 버튼이 자리 잡고 있어요. 중앙에는 검색창과 `새 프로젝트` 버튼, 아래에는 프로젝트 카드가 그리드 형태로 배열돼요.

![프로젝트 대시보드 구성](/images/example.png)

- 검색창은 이름과 설명을 동시에 찾아줘요. 글자를 입력하면 바로 카드 목록이 갱신돼요.
- 카드에는 최근 수정 시간, 설명, 엔티티와 관계 수가 표시돼요.
- 카드 오른쪽 위 점 세 개 버튼을 누르면 `수정`, `삭제` 같은 빠른 메뉴가 열려요.

## 3. 🆕 새 프로젝트 만들기

`새 프로젝트` 버튼을 누르면 작고 아담한 창이 뜨고, 이름과 설명을 적을 수 있어요. 이름은 필수라 비워 두면 "이름을 입력하세요"라는 토스트가, 기존과 동일하면 "중복된 이름"이라는 토스트가 나타나요.

![새 프로젝트 작성 창](/images/example.png)

저장에 성공하면 축하 토스트와 함께 프로젝트가 목록에 추가되고, 곧바로 편집기로 이동해요.

## 4. 🎨 편집기 화면 이해하기

편집기는 전체 화면을 활용해요.

![편집기 기본 레이아웃](/images/example.png)

- **상단 툴바**: 프로젝트 이름과 설명, `엔티티 추가`, `자동 정렬`, `PNG 내보내기`, `스키마 내보내기`, `스키마 가져오기`, `전체 삭제`, `대시보드`, `로그아웃` 등이 순서대로 있어요.
- **왼쪽 상단**: 미니맵과 줌 컨트롤이 있어서 넓은 다이어그램도 쉽게 이동할 수 있어요.
- **우측 하단 플로팅 버튼**: 채팅 기록, 테마 전환, 빠른 엔티티 추가 버튼 세 가지가 동그랗게 떠 있어요.
- **하단 중앙**: AI 어시스턴트에게 말할 수 있는 입력창이 있어요.

## 5. 🧱 첫 테이블 배치하기

상단 툴바에서 `엔티티 추가`를 누르거나 캔버스 빈 곳에서 오른쪽 클릭 → `여기에 엔티티 추가`를 고르면 카드가 생겨요. 엔티티 카드 하나는 곧 테이블이에요.

![새 엔티티 카드 예시](/images/example.png)

- 기본 이름은 `New Entity`예요. 카드 제목을 더블클릭하면 이름을 바로 고칠 수 있어요.
- 첫 컬럼으로 `id`가 만들어져 있고, `INT` 타입과 기본키(각 행을 구분하는 고유 키) 속성이 붙어 있어요.
- 컬럼 옆 연필 아이콘을 누르면 타입(VARCHAR, INT, BOOLEAN, DATE 등), 필수 여부(NOT NULL), 기본값 등을 바꿀 수 있어요.

<Callout type="info">
컬럼 삭제를 누르면 "이 컬럼은 기본키인데 정말 지울까요?"처럼 영향 범위를 알려주는 확인 창이 떠요.
</Callout>

## 6. 🔗 관계(Relationship) 연결하기

두 테이블을 연결하고 싶다면 컬럼 왼쪽의 작은 원을 드래그해서 다른 엔티티로 옮겨보세요. 드래그 중에는 상단에 파란/초록 배지가 떠서 지금이 출발점인지 도착점인지 알려줘요.

![관계 생성 흐름](/images/example.png)

선을 놓으면 **관계 타입 선택 창**이 열려요.

- 1:1, 1:N, 0..1, 0..N 중 하나를 고를 수 있어요.
- "식별 관계" 스위치를 켜면 실선, 끄면 점선으로 표시돼요.

관계 타입을 고르면 **속성 매핑 창**이 이어져요.

- 소스/타깃 엔티티에서 연결할 컬럼을 고르는 메뉴가 있어요.
- 일대다 관계에서는 대상 엔티티에서 외래키(다른 테이블과 연결하는 키) 후보를 자동으로 골라줘요.

## 7. 🤖 AI 어시스턴트에게 도움받기

하단 입력창에 자연어로 "사용자(User) 테이블에 last_login DATE 컬럼을 추가해 줘"처럼 부탁하면 돼요.

![AI 요청과 응답 흐름](/images/example.png)

- Enter를 누르거나 전송 버튼을 클릭하면 메시지가 발송돼요.
- 응답이 오면 우측에서 채팅 사이드바가 열리고, 설명과 함께 JSON 스키마가 카드 형태로 표시돼요.
- JSON이 완전하면 다이어그램이 그 구조로 교체돼요. 마음에 들지 않으면 이전 상태로 되돌릴 수 있도록 다시 요청하기 전에 내보내기로 백업해두면 좋아요.

## 8. ☁️ 자동 저장 확인하기

툴바 오른쪽에는 저장 상태가 `저장 중...`에서 `저장 완료`로 바뀌는 작은 알림이 있어요. 변경 후 약 1초 뒤 서버에 저장되고, 다시 대시보드로 돌아가면 최신 수정 시간이 바로 갱신돼요. 다른 기기에서도 같은 계정으로 로그인하면 그대로 이어서 편집할 수 있어요.

<Callout type="success">
작업을 마쳤다면 상단 `대시보드` 버튼으로 목록으로 돌아가 보세요. 방금 만든 프로젝트 카드가 맨 위에 올라와 있을 거예요.
</Callout>
