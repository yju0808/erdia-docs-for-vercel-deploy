# 💰 가격 정책

Erdia는 지금 당장 시작할 수 있는 **Free 플랜**과, 협업 기능을 강화한 **Pro 플랜(출시 예정)**을 준비하고 있어요. 아래 표로 혜택을 비교해 보고, 팀 상황에 맞는 플랜을 골라보세요.

![가격 카드 UI 예시](/images/example.png)

| 항목 | Free | Pro (예정) |
| --- | --- | --- |
| 월간 요금 | 무료 | 8,900원 |
| 프로젝트 수 | 무제한 | 무제한 |
| AI 채팅 | 50회/일 | 고급 모델, 일 한도 상향 예정 |
| 실시간 편집 | 단일 사용자 | 다중 사용자 동시 편집 |
| 스키마 내보내기 | MySQL · PostgreSQL · Prisma · JPA | 동일 + 버전 관리 |
| 자동 저장/동기화 | 포함 | 포함 |
| 팀 관리 | - | 역할 기반 팀 관리 |

<Callout type="success">
Free 플랜으로 가입해도 신용카드 정보가 필요 없어요. 먼저 기능을 충분히 체험한 뒤 Pro 플랜이 열리면 손쉽게 업그레이드할 수 있어요.
</Callout>

---

## 🆓 Free 플랜에서 누릴 수 있는 것

![Free 플랜 체크리스트](/images/example.png)

- [x] Google 로그인만으로 즉시 프로젝트 생성
- [x] 엔티티 편집, 관계 생성, 자동 정렬 등 모든 캔버스 기능
- [x] MySQL · PostgreSQL · Prisma · JPA 내보내기와 가져오기
- [x] 하루 50회의 AI 채팅으로 초안 생성 및 수정 제안
- [x] 실시간 자동 저장과 대시보드에서의 최근 수정 시간 업데이트

<Callout type="info">
하루 AI 채팅 한도는 매일 자정(프로젝트 설정의 로컬 시간 기준) 이후 자동으로 초기화돼요. 설계를 많이 시도할 계획이라면 한도를 염두에 두고 메시지를 구성해 주세요.
</Callout>

---

## 🚀 Pro 플랜 미리보기

![Pro 플랜 로드맵](/images/example.png)

- 실시간 협업: 여러 명이 동시에 같은 다이어그램을 편집하고, 커서 위치를 서로 확인할 수 있어요.
- 고급 AI: 더 긴 맥락과 복잡한 스키마를 이해하는 모델과 추가 토큰 한도를 제공할 예정이에요.
- 버전 관리: 주요 시점을 스냅샷으로 남겨 변경 이력을 추적하고, 필요하면 되돌릴 수 있어요.
- 팀 관리: 역할별 권한(예: Viewer, Editor)을 설정해 안정적으로 협업할 수 있게 준비 중이에요.

<Callout type="warning">
Pro 플랜은 아직 출시 전이라 결제 기능이 비활성화돼 있어요. 출시 소식을 빠르게 받고 싶다면 [support@erdia.app](mailto:support@erdia.app)으로 연락해 주세요.
</Callout>

---

## ⏱ 업그레이드를 고려할 타이밍

1. Free 플랜의 AI 채팅 50회/일 한도가 자주 부족할 때
2. 팀원이 동시에 편집하거나 변경 이력을 추적해야 할 때
3. 테이블 버전을 비교하며 협업 리뷰를 진행하고 싶을 때

![업그레이드 배너](/images/example.png)

<Callout type="info">
Pro 플랜은 출시와 동시에 안내 모달이 대시보드에 표시될 예정이에요. 알림을 놓치지 않으려면 로그인 상태를 유지하고, 이메일 공지를 확인해 주세요.
</Callout>
