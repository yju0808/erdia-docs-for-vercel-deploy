# 💰 가격 정책

지금 바로 사용할 수 있는 **Free 플랜**과, 준비 중인 **Pro 플랜**을 비교해보세요.

![가격 플랜](/images/example.png)

---

## 📊 플랜 비교

| 기능 | Free | Pro (예정) |
| --- | --- | --- |
| 월 요금 | 무료 | 8,900원 |
| 프로젝트 | 무제한 | 무제한 |
| AI 채팅 | 50회/일 | 고급 모델 + 증가 |
| 동시 편집 | 단일 사용자 | 다중 사용자 |
| 내보내기 | MySQL, PostgreSQL<br/>Prisma, JPA | 동일 + 버전 관리 |
| 팀 관리 | - | 역할 기반 권한 |

---

## 🆓 Free 플랜

**포함 기능**
- [x] Google 로그인으로 즉시 시작
- [x] ERD 편집 (테이블, 컬럼, 관계)
- [x] MySQL, PostgreSQL, Prisma, JPA 내보내기
- [x] AI 채팅 50회/일
- [x] 자동 저장 및 동기화

**시작 방법**
- 신용카드 등록 없이 바로 사용
- 홈에서 "무료로 시작하기" 클릭

<Callout type="info">
AI 채팅 한도는 매일 자정에 초기화돼요. 한도를 효율적으로 사용하려면 구체적으로 요청하세요.
</Callout>

---

## 🚀 Pro 플랜 (준비 중)

**추가 기능**
- 실시간 다중 사용자 편집
- 고급 AI 모델
- 버전 관리 및 이력 추적
- 팀 권한 관리 (Viewer, Editor 등)

**출시 예정**
- 현재 개발 중이에요
- 출시 알림을 받으려면 [support@erdia.app](mailto:support@erdia.app)으로 문의하세요
