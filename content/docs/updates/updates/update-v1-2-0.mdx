## ✨ 새로운 기능

### PostgreSQL JSONB 타입 지원

이제 PostgreSQL 내보내기 시 JSONB 데이터 타입을 사용할 수 있습니다!

```sql
CREATE TABLE User (
  id SERIAL PRIMARY KEY,
  email VARCHAR(255) NOT NULL,
  metadata JSONB  -- 새로운 JSONB 타입!
);
```

**JSONB의 장점:**
- 빠른 쿼리 성능
- 인덱싱 지원
- 효율적인 저장 공간

**사용 예시:**

```sql
-- JSONB 데이터 삽입
INSERT INTO User (email, metadata) VALUES (
  'user@example.com',
  '{"preferences": {"theme": "dark", "language": "ko"}}'::jsonb
);

-- JSONB 쿼리
SELECT * FROM User 
WHERE metadata->>'theme' = 'dark';
```
