# 🎯 기능 한눈에 보기

Erdia는 프로젝트 관리, 비주얼 편집, AI 어시스턴트, 가져오기·내보내기 네 가지 축으로 구성됩니다. 아래 표에서 주요 기능과 제공 가치를 확인하세요.

| 범주 | 주요 기능 | 사용자 혜택 |
| --- | --- | --- |
| 프로젝트 관리 | 대시보드 검색, 프로젝트 생성·수정·삭제, 최신순 정렬 | 여러 설계안을 안전하게 보관하고 빠르게 찾아 다시 열 수 있습니다. |
| 비주얼 편집 | 엔티티/속성/관계 편집, 컨텍스트 메뉴, 자동 정렬, 테마 토글 | 복잡한 다이어그램도 보기 좋게 정리하고, 반복 작업을 최소화할 수 있습니다. |
| AI 어시스턴트 | 자연어 요청, JSON 스키마 자동 감지, 채팅 사이드바 | 텍스트 설명만으로 초안을 만들고, 응답 기록을 보면서 빠르게 다듬을 수 있습니다. |
| 데이터 교환 | MySQL·PostgreSQL·Prisma·JPA 내보내기, 동일 형식 가져오기 | 기존 시스템과의 연결이 쉬워지고, 협업 팀에 맞는 포맷으로 즉시 공유할 수 있습니다. |

## ✨ 사용자 경험 원칙

- **즉시성** – 버튼을 누르는 즉시 토스트와 다이얼로그로 결과를 알려 드립니다.
- **복구 가능성** – 삭제 전에는 영향 범위를 알려주는 확인 창이 항상 표시됩니다.
- **가시성** – 자동 저장 상태, 관계 연결 가이드, 토글 버튼 등 중요한 피드백을 즉각 제공합니다.
