# 🎯 주요 기능

ERD 설계부터 코드 생성까지, 개발 흐름에 맞춰 필요한 기능을 정리했어요.

![주요 기능 미리보기](/images/example.png)

---

## 🗂 프로젝트 관리

**검색과 필터링**
- 이름이나 설명으로 빠르게 찾을 수 있어요
- 최근 수정 순으로 자동 정렬돼요

**생성과 수정**
- `새 프로젝트` 버튼으로 즉시 생성
- 이름 중복 검사로 실수 방지
- 카드에서 바로 수정/삭제 가능

<Callout type="info">
프로젝트를 생성하거나 수정하면 토스트로 결과가 즉시 표시돼요. 실패 메시지에 적힌 내용을 참고하면 입력 누락이나 중복 이름을 바로 해결할 수 있습니다.
</Callout>

---

## 🎨 ERD 편집

**테이블 추가**
- 버튼 클릭이나 우클릭으로 추가
- 더블클릭으로 이름 변경
- 기본 `id` 컬럼 자동 생성

**컬럼 편집**
- 타입, 기본키, 외래키, NULL 여부 설정
- 삭제 시 영향 범위 경고
- 드래그로 순서 변경

**관계 설정**
- 드래그로 테이블 연결
- 1:1, 1:N 등 관계 타입 선택
- 외래키 자동 매핑

**편의 기능**
- 자동 정렬로 겹침 해소
- 미니맵과 줌으로 전체 탐색
- 1초 이내 자동 저장

## 🤖 AI 어시스턴트

**자연어 요청**
- "쇼핑몰 ERD 만들어줘" 처럼 간단하게 요청
- 기존 ERD 수정도 가능
- 하루 50회 무료 사용

**스마트 응답**
- 한국어 설명 + 코드 제공
- 완성된 구조 자동 반영
- 대화 기록 저장

<Callout type="warning">
AI로 만든 구조를 적용하면 이전 상태로 되돌릴 수 없어요. 중요한 작업은 먼저 백업하세요.
</Callout>

## 📤 내보내기 & 공유

**지원 포맷**
- MySQL, PostgreSQL
- Prisma, JPA

**옵션 선택**
- 외래키, 인덱스 포함 여부
- 타임스탬프 자동 추가
- 스키마 이름 지정

**간편한 공유**
- 코드 복사 또는 파일 다운로드
- 팀 규칙에 맞춰 커스터마이징

<Callout type="info">
PNG 내보내기 버튼은 아직 구현 전이에요. 이미지 공유가 필요하면 브라우저 캡처 기능을 이용해 주세요.
</Callout>
