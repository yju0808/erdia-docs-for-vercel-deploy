# 🎯 기능 한눈에 보기

Erdia는 "아이디어를 ERD로 옮긴 뒤 팀과 공유한다"라는 여정을 중심으로 설계됐어요. 각 단계에서 어떤 도구가 여러분을 도와주는지 간단히 훑어볼게요.

![Erdia 여정 요약](/images/example.png)

## 1. 🗂 프로젝트 정리하기

| 할 일 | 도움이 되는 기능 | 얻을 수 있는 가치 |
| --- | --- | --- |
| 여러 설계 버전을 구분하고 싶어요 | 대시보드 목록, 검색, 최근 수정순 정렬 | 필요한 다이어그램을 즉시 찾고 다시 열 수 있어요 |
| 새 아이디어를 시험해 보고 싶어요 | `새 프로젝트` 생성 창, 중복 이름 감지 | 잘못된 이름으로 덮어쓰는 실수를 예방해요 |
| 팀원에게 상태를 공유하고 싶어요 | 카드에 표시되는 설명과 최근 수정 시간 | 누구나 지금 상태를 빠르게 이해할 수 있어요 |

## 2. 🎨 캔버스에서 다듬기

| 할 일 | 기능 | 가치 |
| --- | --- | --- |
| 새 테이블(엔티티)을 추가하고 싶어요 | `엔티티 추가`, 컨텍스트 메뉴, 더블클릭 이름 편집 | 손이 기억하는 방식으로 빠르게 구조를 만들 수 있어요 |
| 컬럼을 편집하거나 제약조건을 바꾸고 싶어요 | 컬럼 편집 패널, 기본키·외래키 토글 | 정확한 데이터 타입과 제약조건을 설정해 오류를 줄여요 |
| 관계(Relationship)를 명확히 표현하고 싶어요 | 관계 타입 선택, 식별/비식별 토글, 컬럼 매핑 | 테이블 간 연결 방식이 누구에게나 명확해져요 |
| 보기 좋은 배치가 필요해요 | 자동 정렬, 미니맵, 줌 컨트롤 | 복잡한 다이어그램도 한눈에 파악할 수 있어요 |

## 3. 🤖 AI에게 초안 부탁하기

| 할 일 | 기능 | 가치 |
| --- | --- | --- |
| 머릿속 구조를 빠르게 시각화하고 싶어요 | 채팅 입력창, 자동 JSON 반영 | 문장 몇 줄로 다이어그램을 만들어 초안을 빠르게 확인할 수 있어요 |
| 기존 구조에 변화를 제안받고 싶어요 | `현재 ERD 상태:` 프롬프트, 채팅 사이드바 기록 | 지금 상태를 기준으로 AI가 수정안을 제안해 줘서 실험이 쉬워요 |

## 4. 📤 내보내고 공유하기

| 할 일 | 기능 | 가치 |
| --- | --- | --- |
| 개발팀과 스키마를 공유하고 싶어요 | MySQL·PostgreSQL·Prisma·JPA 내보내기 | 팀이 사용하는 포맷으로 바로 전달할 수 있어요 |
| 기존 시스템을 불러오고 싶어요 | 동일 형식 가져오기, 외래키 포함 옵션 | 현재 서비스 구조를 그대로 캔버스로 옮겨 비교할 수 있어요 |
| 그림으로 공유하고 싶어요 | PNG 내보내기 버튼(업데이트 예정) | 캔버스를 이미지로 추출할 예정이에요 |
