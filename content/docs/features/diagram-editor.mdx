# 🎨 캔버스 & 편집기

Erdia 편집기는 눈앞에서 구조를 조립하듯 ERD를 만들 수 있게 도와줘요. 여기서는 테이블(=엔티티)과 컬럼(=테이블 안의 칸), 관계(Relationship)를 어떻게 다루는지 하나씩 알아볼게요.

## 🧱 화면 기본 구조

![편집기 기본 구성](/images/example.png)

- **상단 툴바**: 프로젝트 이름과 설명, `엔티티 추가`, `자동 정렬`, `PNG 내보내기`, `스키마 내보내기`, `스키마 가져오기`, `전체 삭제`, `대시보드`, `로그아웃` 버튼이 모여 있어요.
- **왼쪽 상단**: 미니맵과 줌 컨트롤이 있어서 넓은 다이어그램도 쉽게 둘러볼 수 있어요.
- **우측 하단 플로팅 버튼**: 채팅 기록 열기, 테마 전환, 빠른 엔티티 추가 버튼이 동그랗게 떠 있어요.
- **하단 중앙**: AI 어시스턴트에게 말을 걸 수 있는 입력창이 자리잡고 있어요.

<Callout type="info">
툴바 오른쪽에는 `저장 중...` → `저장 완료` 배지가 보이는데, 이게 자동 저장 상태를 알려줘요.
</Callout>

## ➕ 엔티티(테이블) 추가하고 이름 바꾸기

![엔티티 추가 흐름](/images/example.png)

1. 상단 툴바의 `엔티티 추가` 버튼을 누르거나, 캔버스 빈 공간에서 오른쪽 클릭 → `여기에 엔티티 추가`를 선택해요.
2. `New Entity` 카드가 생기고, 기본 컬럼으로 `id`가 들어 있어요.
3. 카드 제목을 더블클릭하면 이름 편집 모드가 열려요. 엔터를 치거나 입력창 밖을 클릭하면 저장돼요.
4. 카드 우측 상단 휴지통 아이콘은 전체 엔티티 삭제예요. 누르면 확인 창이 떠서 실수를 방지해요.

<Callout type="success">
엔티티를 많이 만들었다면 자동 정렬을 눌러 한 번에 보기 좋게 정렬할 수 있어요. 아래에서 더 자세히 알아볼게요.
</Callout>

엔티티를 우클릭하면 작은 메뉴가 떠요. 여기에서 `엔티티 편집`을 눌러 즉시 이름을 바꾸거나, `엔티티 삭제`로 빠르게 정리할 수 있어요.

## ✏️ 컬럼 다듬기

![컬럼 편집 패널](/images/example.png)

- 각 컬럼 오른쪽의 연필 아이콘을 누르면 작은 편집 패널이 열려요.
- 타입은 VARCHAR, INT, BOOLEAN, DATE 등으로 바꿀 수 있고, 기본값도 지정할 수 있어요.
- 체크박스로 기본키(각 행을 구분하는 고유 키), 외래키(다른 테이블과 연결하는 키), NOT NULL 같은 제약조건을 조절할 수 있어요.
- 컬럼 삭제를 누르면 역할에 따라 경고 메시지가 함께 표시돼요.

컬럼을 새로 만들고 싶다면 카드 하단 `+ 속성 추가` 버튼을 눌러 이름을 적으면 돼요. 입력 중 엔터를 누르면 바로 저장되고, 취소하려면 ESC를 누르면 돼요.

## 🔗 관계 만들고 조정하기

![관계 생성과 설정](/images/example.png)

1. 연결하고 싶은 컬럼 왼쪽의 작은 원(핸들)을 드래그해서 다른 엔티티로 옮겨요.
2. 드래그 중에는 상단에 파란/초록 배지가 떠서 지금이 출발인지 도착인지 알려줘요.
3. 선을 놓으면 관계 타입 선택 창이 열려요. 1:1, 1:N, 0..1, 0..N 중에서 고르고, 식별 관계 스위치를 켜면 실선, 끄면 점선으로 표시돼요.
4. 이어서 속성 매핑 창이 열려요. 소스·타깃 엔티티의 컬럼을 선택하거나, 엔티티 전체 관계로 둘 수도 있어요. 일대다 관계에서는 타깃 엔티티의 외래키 후보를 자동으로 추천해줘요.
5. 관계 레이블 옆의 톱니바퀴 아이콘을 누르면 팝오버가 열리고, 관계 타입과 식별 여부를 다시 조정하거나 `삭제` 버튼으로 선을 지울 수 있어요.

<Callout type="warning">
관계를 만들다가 마음이 바뀌면 ESC를 눌러 연결 모드를 종료하세요. 연결 가이드 배지가 사라지면서 초기화돼요.
</Callout>

## 🧰 툴바 활용 팁

![툴바 버튼 모음](/images/example.png)

- **자동 정렬**: 버튼을 누르면 엔티티가 서로 겹치지 않도록 재배치되고, 곧바로 전체 화면에 맞춰 줌이 조정돼요.
- **PNG 내보내기**: 아직 준비 중인 버튼이에요. 업데이트가 되면 캔버스를 이미지로 저장할 수 있게 될 거예요.
- **스키마 내보내기 / 가져오기**: MySQL, PostgreSQL, Prisma, JPA 네 가지 형식을 지원해요. 자세한 사용법은 [🔄 가져오기 · 내보내기](/docs/features/import-export)에서 확인하세요.
- **전체 삭제**: 다이어그램을 싹 비우는 기능이에요. 클릭하면 영향 범위를 안내하는 확인 창이 먼저 뜹니다.

## 🌗 테마와 플로팅 버튼

![플로팅 버튼 모음](/images/example.png)

- 말풍선 버튼: 채팅 기록 사이드바를 켜고 끌 수 있어요.
- 태양/달 버튼: 라이트/다크 테마를 전환해요. 한 번 고르면 이후에도 동일한 테마가 유지돼요.
- `+` 버튼: 화면 어디서든 새 엔티티를 바로 추가해요.

## ⌨️ 자주 쓰는 단축키

| 단축키 | 설명 |
| --- | --- |
| `Delete` / `Backspace` | 선택한 엔티티 또는 관계를 삭제해요 (항상 확인 창이 떠요). |
| `Esc` | 열려 있는 메뉴나 편집 모드를 닫아요. |
| `Enter` | 엔티티 이름/컬럼 입력을 저장해요. |

<Callout type="success">
단축키는 입력창이나 확인 창이 열려 있지 않을 때만 동작해요. 실수로 삭제하지 않도록 안전장치가 곳곳에 숨겨져 있어요.
</Callout>
