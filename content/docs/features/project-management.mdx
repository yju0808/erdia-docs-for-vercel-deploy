# 📁 프로젝트 관리

대시보드는 Erdia 여정의 출발점이에요. 로그인만 하면 모든 프로젝트가 서버에서 불러와지고, 가장 최근에 다듬은 순서대로 정렬돼요. 여기서는 프로젝트를 빠르게 찾고 정리하는 방법을 함께 살펴볼게요.

![대시보드 미리보기](/images/example.png)

## 🔍 필요한 프로젝트를 바로 찾기

1. 상단 검색창에 키워드를 입력해 보세요.
2. 입력과 동시에 카드 목록이 업데이트되고, 이름과 설명을 모두 찾아줘요.
3. 검색 결과가 없으면 "검색 결과가 없습니다" 카드가 나타나 새 프로젝트로 바로 이동하도록 안내해요.

![검색과 필터링 예시](/images/example.png)

<Callout type="info">
검색어를 지우면 전체 목록이 다시 보이고, 항상 최근 수정 순으로 정렬돼요. 최신 작업물을 놓칠 걱정이 없어요.
</Callout>

## 🆕 새 프로젝트 만들기

1. 검색창 오른쪽에 있는 `새 프로젝트` 버튼을 눌러요.
2. 작고 아담한 창에서 이름과 설명을 입력해요. 이름은 필수고, 설명은 선택이에요.
3. 이름이 비어 있거나 기존 프로젝트와 겹치면 토스트가 나타나 알려줘요.
4. 저장에 성공하면 축하 토스트와 함께 바로 편집기로 이동해요.

![새 프로젝트 창과 토스트](/images/example.png)

<Callout type="success">
생성한 순간부터 프로젝트는 서버에 저장돼요. 다른 기기에서도 같은 계정으로 로그인하면 바로 목록에 나타나요.
</Callout>

## ✏️ 프로젝트 정보 수정하기

1. 카드 오른쪽 위 점 세 개 버튼을 눌러요.
2. `수정`을 선택하면 동일한 입력 창이 열리고, 이름과 설명을 바꿀 수 있어요.
3. 저장하면 `프로젝트 수정 완료` 토스트가 뜨고, 카드 정보가 즉시 업데이트돼요.

![프로젝트 카드 메뉴](/images/example.png)

<Callout type="info">
이름을 바꾸면 목록의 정렬 순서도 최신 순으로 다시 올라와요. 최근에 손댄 설계가 항상 위로 올라온답니다.
</Callout>

## 🗑️ 프로젝트 삭제하기

1. 동일한 메뉴에서 `삭제`를 선택해요.
2. 삭제 전 확인 창이 열리고, 취소/삭제 중 하나를 고를 수 있어요.
3. 삭제를 승인하면 카드가 사라지고, "프로젝트 삭제 완료" 토스트가 표시돼요.

![프로젝트 삭제 확인 창](/images/example.png)

<Callout type="warning">
삭제하면 되돌릴 수 없어요. 중요한 다이어그램이라면 먼저 [🔄 가져오기 · 내보내기](/docs/features/import-export)에서 스키마를 백업해 두세요.
</Callout>

## 🔄 대시보드로 돌아오는 길

편집기 상단 툴바의 `대시보드` 버튼을 누르면 언제든 목록으로 돌아올 수 있어요. 자동 저장이 끝난 뒤 돌아오면 카드의 최근 수정 시간이 갱신된 걸 바로 확인할 수 있어요.

<Callout type="success">
작업을 마칠 때 `로그아웃`을 눌러 세션을 정리할 수 있어요. 로그아웃하면 액세스 토큰이 비워지고, 다음에 방문했을 때 다시 로그인하라는 안내가 나타나요.
</Callout>
