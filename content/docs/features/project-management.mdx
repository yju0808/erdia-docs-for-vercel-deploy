# 📁 프로젝트 관리

로그인하면 가장 먼저 대시보드가 열려요. 여기에서 프로젝트를 검색하고 새로 만들며, 필요할 때는 수정하거나 삭제할 수 있습니다. 아래 순서를 참고하면 업무 중 원하는 다이어그램을 빠르게 찾고 정리할 수 있어요.

![대시보드 미리보기](/images/example.png)

---

## 🔍 필요한 프로젝트를 바로 찾기

1. 상단 검색창에 키워드를 입력하면 이름과 설명을 동시에 검색해요.
2. 입력과 동시에 카드 목록이 갱신되고, 결과가 없으면 빈 상태 카드가 나타나 새 프로젝트 생성으로 안내합니다.
3. 검색어를 지우면 전체 목록이 다시 보이고, 항상 최근 수정 순으로 정렬돼요.

<Callout type="info">
검색이 실패하거나 서버 오류가 발생하면 토스트로 이유가 함께 표시돼요. 메시지를 참고해 네트워크 상태를 확인하거나 잠시 후 다시 시도해 주세요.
</Callout>

---

## 🆕 새 프로젝트 만들기

1. 검색창 오른쪽 **새 프로젝트** 버튼을 눌러요.
2. 이름과 설명을 입력합니다. 이름은 필수고, 설명은 선택이에요.
3. 저장하면 성공 토스트와 함께 편집기로 이동하고, 목록에는 새 카드가 즉시 추가됩니다.

![새 프로젝트 창과 토스트](/images/example.png)

<Callout type="warning">
이름을 비워 두거나 기존 프로젝트와 같게 입력하면 토스트로 오류가 안내돼요. 이름 뒤에 버전이나 날짜를 붙여 구분해 주세요.
</Callout>

---

## ✏️ 프로젝트 정보 수정하기

1. 카드 오른쪽 위 점 세 개 버튼을 눌러 컨텍스트 메뉴를 열어요.
2. **수정**을 선택하면 생성 때와 동일한 입력 창이 열립니다.
3. 저장하면 카드 정보가 즉시 업데이트되고, 최근 수정 순 정렬 덕분에 목록 상단으로 이동해요.

![프로젝트 카드 메뉴](/images/example.png)

---

## 🗑️ 프로젝트 삭제하기

1. 같은 메뉴에서 **삭제**를 선택하면 확인 다이얼로그가 열려요.
2. 삭제를 승인하면 카드가 사라지고, 성공 토스트가 표시됩니다.
3. 삭제는 되돌릴 수 없으니, 중요한 다이어그램이라면 먼저 내보내기로 백업해 두세요.

![프로젝트 삭제 확인 창](/images/example.png)

<Callout type="warning">
삭제가 실패하면 메시지에 표시된 사유(권한 부족, 네트워크 오류 등)를 확인하고 다시 시도하세요. 문제가 반복되면 백엔드 로그 확인을 요청해 주세요.
</Callout>

---

## 🔄 대시보드로 돌아오기

편집기 상단 툴바의 **대시보드** 버튼을 누르면 언제든 목록으로 돌아올 수 있어요. 자동 저장이 끝난 상태에서 이동하면 카드의 최근 수정 시간이 업데이트된 걸 바로 확인할 수 있습니다.

<Callout type="success">
로그아웃을 누르면 액세스 토큰이 초기화돼요. 공용 기기에서 작업했다면 꼭 로그아웃해 주세요.
</Callout>

---

## 🔁 다른 작업으로 전환하기

대시보드에서 원하는 프로젝트를 열었다면, 편집기 오른쪽 상단 **내보내기** 버튼으로 바로 SQL이나 코드 템플릿을 생성할 수 있어요. 작업을 마친 뒤에는 같은 위치의 **공유** 메뉴로 팀과 다이어그램 링크를 전달해 주세요.

<Callout type="info">
대규모 자동화용 API는 준비 중이에요. 현재는 내보내기 결과를 Git 저장소나 위키에 저장해 두고, 변경 사항이 생길 때마다 새로 내보내어 반영하는 흐름을 추천해요.
</Callout>
